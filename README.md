# SkillHire Website

A smart role-based job portal web application developed using HTML, CSS, and JavaScript.

## Features

### Student
- User registration and login
- Browse job listings
- Search and filter jobs
- Apply for jobs
- Track applied jobs with application status
- Create and manage portfolio profile
- View recommended jobs based on skills
- Dark mode support
- Mobile responsive UI

### Recruiter
- Recruiter authentication
- Post job openings
- View applicants per job
- View student profiles
- Update application status (Applied / Reviewed / Selected)

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Browser LocalStorage

## Project Highlights
- Role-based access control
- Skill-based job recommendation system
- Responsive and dark mode UI
- Fully frontend implementation (no backend)

## Future Enhancements
- Backend integration
- Resume parsing
- AI-based job matching
- Admin dashboard

## How to Run
Simply open `index.html` in a browser.
